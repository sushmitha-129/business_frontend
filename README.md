# 🧵 Sudha Textiles - Saree E-Commerce Website

A modern e-commerce web application for browsing and purchasing cotton sarees. Built with a clean UI and dynamic product display using React and MongoDB.

---

## 🚀 Features

- 🛍️ Browse saree products
- 🔍 Search functionality
- 🖼️ Product images from database
- 📄 Product details page
- 🛒 Add to Cart
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- JavaScript (ES6)
- HTML5, CSS3

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB Atlas
